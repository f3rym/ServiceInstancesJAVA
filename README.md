### ☁️ Service Instances JAVA (Project F3Cloud)
**Разработка облачной платформы для аренды вычислительных ресурсов (REST API)**

### О проекте

Учебный REST-сервис на Spring Boot для управления инстансами облачной платформы. Выполнен в рамках лабораторной работы по дисциплине "Программирование на языках высокого уровня".

### Функциональность

Проект реализует базовые операции REST API для сущности `CloudInstance` (виртуальный сервер):

*   **Получение инстанса по ID** — GET-запрос с использованием `@PathVariable`.
*   **Фильтрация списка инстансов** — GET-запрос с использованием `@RequestParam` (по типу и статусу).
*   **Слоистая архитектура** — четкое разделение на Controller, Service и Repository слои.
*   **Использование DTO и Mapper** — изоляция данных сущности от клиента.
*   **Обработка ошибок** — кастомное исключение `InstanceNotFoundException` с HTTP статусом 404.
*   **Статический анализ кода** — интеграция с Checkstyle для поддержания единого стиля кодирования.

### Используемые технологии

*   **Java 17+** — язык программирования.
*   **Spring Boot 3** — основной фреймворк для создания REST API.
*   **Spring Web** — для создания REST-контроллеров.
*   **Lombok** — для уменьшения шаблонного кода.
*   **Maven** — система сборки и управления зависимостями.
*   **Checkstyle** — инструмент статического анализа кода.
